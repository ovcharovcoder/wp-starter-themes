A modern WordPress block-based starter theme for converting HTML landing pages into Gutenberg-ready WordPress sites.

== Description ==

WP Block Starter is a flexible, developer-friendly starter theme built around the WordPress block editor (Gutenberg).

Features:
* Fully block-based templates in /templates/
* Modular sections and reusable components in /parts/
* CSS reset via optimize.css and main.css for custom styling
* editor.css for styling blocks in Gutenberg editor
* JS integration ready for interactivity
* Responsive design out-of-the-box
* Supports custom patterns via /patterns/
* Structured theme setup via /inc/

This theme is perfect for developers who want to quickly turn static landing pages into fully editable block-based WordPress sites.

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`.
2. Activate the theme in WordPress admin.
3. All block templates are in `/templates/` and can be edited or extended.
4. Place your assets (images, CSS, JS) in `/assets/`.
5. Register new patterns by adding PHP files to `/patterns/` if needed.
