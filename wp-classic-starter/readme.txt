A clean starter classic WordPress theme for converting HTML landing pages into WordPress.

== Description ==

WP Classic Theme is a minimal yet flexible WordPress theme designed for developers who want to quickly
turn HTML landing pages into fully functional WordPress sites.

Features:
* Minimal PHP template structure
* Supports custom page templates in /templates/
* Modular reusable sections in /parts/
* Optimize.css for CSS reset
* Main.css for custom styling
* JS integration ready
* Fully responsive and developer-friendly

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`.
2. Activate the theme in WordPress admin.
3. Create pages and assign custom templates if needed (from /templates/).
4. Place your images in `/assets/images/` and styles in `/assets/css/`.
